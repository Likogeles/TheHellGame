Запуск через "HellGameEXE.bat" (может не работать)
	(Усердно работаю над созданием .exe файла для игры, но пока только так)

Запуск возможен только через PyCharm через файл "main.py"

Необходимы библиотеки:
	pygame
	time
	os
	sqlite3
	sys
	math
Файлы:
	data содержит карты уровней, музыку, звуки и спрайты
	classes.py - содержит классы
	dialogues.py - содержит функции диалогов
	functions.py - содержит функции
	main.py - содержит основной скрипт
	save.db - база данных сохранения
	scenes.py - содержит сцены игры
	HellGameEXE.bat - программа запуска (временно заменяет .exe файл, я работа над этим)
P.S.
	На видео и в прочих документах устаревшие версии игры, актуальная версия всегда тут: https://github.com/Likogeles/TheHellGame
	Видео тормозит из-за большой загруженности ПК, но игра достаточно оптимизирована
